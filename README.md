# Projekt amortyzatora samochodowego – CAD (Siemens NX)

Repozytorium zawiera model 3D amortyzatora samochodowego wykonany w programie **Siemens NX** w ramach projektu realizowanego na studiach kierunku **Mechanika i Projektowanie Maszyn**.

## Zakres projektu

Projekt obejmował:

- modelowanie 3D poszczególnych elementów amortyzatora
- stworzenie złożenia (assembly) całego układu
- przygotowanie dokumentacji technicznej i rysunków wykonawczych
- modelowanie deformującej się sprężyny zawieszenia
- przygotowanie widoku eksplodowanego zespołu

Model przedstawia konstrukcję typowego amortyzatora stosowanego w układach zawieszenia pojazdów.

## Główne elementy konstrukcji

Model amortyzatora składa się z wielu części, m.in.:

- korpus amortyzatora
- tłok
- tłoczysko
- wkład (cartridge)
- sprężyna zawieszenia
- talerz sprężyny
- nakrętka regulacyjna sprężyny
- końcówka kulowa
- pierścień zabezpieczający
- dystans wkładu

Dodatkowo w repozytorium znajdują się pliki przedstawiające:

- złożenie amortyzatora
- widok eksplodowany
